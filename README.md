# Recommender System - Movie Lens Dataset
<hr/>


Dataset from [kaggle](https://www.kaggle.com/datasets/rounakbanik/the-movies-dataset)

## Install both python and npm packages
```shell
npm install-all
```

## Run Prod App
```shell
npm start
```

## Run Dev 
```shell
npm run dev 
```

## Build for prod 
```shell
npm run build 
```
